🔐 Login & Registration Page

A responsive and animated Login & Registration UI built using HTML, CSS, and JavaScript.

🚀 Features

Glassmorphism design

Smooth toggle animation between Login and Registration

Floating label input fields

Responsive layout

Clean navigation bar with hover effects

Popup modal interaction

🛠 Tech Stack

HTML5

CSS3 (Flexbox, Transitions, Transform)

JavaScript (DOM Manipulation)

⚙️ How It Works

JavaScript toggles classes on .wrapper

active class switches between login and register

active-popup controls popup visibility

CSS handles animations and transitions
